							<p>This example reads the contents of a text file by using the static methods <a href="/en-us/dotnet/api/system.io.file.readalltext" data-linktype="absolute-path">ReadAllText</a> and <a href="/en-us/dotnet/api/system.io.file.readalllines" data-linktype="absolute-path">ReadAllLines</a> from the <a href="/en-us/dotnet/api/system.io.file" data-linktype="absolute-path">System.IO.File</a> class.</p>
<p>For an example that uses <a href="/en-us/dotnet/api/system.io.streamreader" data-linktype="absolute-path">StreamReader</a>, see <a href="how-to-read-a-text-file-one-line-at-a-time" data-linktype="relative-path">How to read a text file one line at a time</a>.</p>
<div class="NOTE">
<p>Note</p>
<p>The files that are used in this example are created in the topic <a href="how-to-write-to-a-text-file" data-linktype="relative-path">How to write to a text file</a>.</p>
</div>
<h2 id="example">Example</h2>
<pre><code class="lang-csharp" name="csFilesandFolders#4">class ReadFromFile
{
    static void Main()
    {
        // The files used in this example are created in the topic
        // How to: Write to a Text File. You can change the path and
        // file name to substitute text files of your own.

        // Example #1
        // Read the file as one string.
        string text = System.IO.File.ReadAllText(@&quot;C:\Users\Public\TestFolder\WriteText.txt&quot;);

        // Display the file contents to the console. Variable text is a string.
        System.Console.WriteLine(&quot;Contents of WriteText.txt = {0}&quot;, text);

        // Example #2
        // Read each line of the file into a string array. Each element
        // of the array is one line of the file.
        string[] lines = System.IO.File.ReadAllLines(@&quot;C:\Users\Public\TestFolder\WriteLines2.txt&quot;);

        // Display the file contents by using a foreach loop.
        System.Console.WriteLine(&quot;Contents of WriteLines2.txt = &quot;);
        foreach (string line in lines)
        {
            // Use a tab to indent each line of the file.
            Console.WriteLine(&quot;\t&quot; + line);
        }

        // Keep the console window open in debug mode.
        Console.WriteLine(&quot;Press any key to exit.&quot;);
        System.Console.ReadKey();
    }
}
</code></pre><h2 id="compiling-the-code">Compiling the Code</h2>
<p>Copy the code and paste it into a C# console application.</p>
<p>If you are not using the text files from <a href="how-to-write-to-a-text-file" data-linktype="relative-path">How to write to a text file</a>, replace the argument to <code>ReadAllText</code> and <code>ReadAllLines</code> with the appropriate path and file name on your computer.</p>
<h2 id="robust-programming">Robust Programming</h2>
<p>The following conditions may cause an exception:</p>
<ul>
<li>The file doesn't exist or doesn't exist at the specified location. Check the path and the spelling of the file name.</li>
</ul>
<h2 id="net-security">.NET Security</h2>
<p>Do not rely on the name of a file to determine the contents of the file. For example, the file <code>myFile.cs</code> might not be a C# source file.</p>
